Understanding Event Studies

How to measure the market impact of climate policy announcements.

The Event Study Method

An event study asks: "Did this event change stock prices beyond what we'd normally expect?"

1Estimate "normal" returns
2Observe actual returns
3Subtract: Abnormal Return
4Accumulate (CAR)
Abnormal Return = Actual Return − Expected Return (from market model)
CAR(t₁, t₂) = Sum of daily abnormal returns from day t₁ to t₂
